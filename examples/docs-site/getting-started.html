Getting Started

Create a site in the Dashboard, then embed the widget with your Site ID.

Install

<script src="https://unpkg.com/fidbak@latest/dist/fidbak.min.js"></script>

Usage

<script src="https://unpkg.com/fidbak@latest/dist/fidbak.min.js"></script>
        <script>
          Fidbak.init({
            siteId: 'YOUR_SITE_ID',
            theme: 'auto',
            position: 'br',
            includeQuery: false,
            captureSelection: true
          });
        
Next Steps
