Getting Started

Install the widget from npm and initialize it with your Site ID.

Install

npm i @fidbak/widget
# or
pnpm add @fidbak/widget

Usage

import fidbak from '@fidbak/widget';

fidbak('init', {
  siteId: 'YOUR_SITE_ID',
  theme: 'auto',
  position: 'br',
  draggable: true,
  includeQuery: false,
  captureSelection: true,
  apiBaseUrl: 'https://fidbak-api.primary-account-45e.workers.dev',
  webhookUrl: ['https://hooks.slack.com/services/XXX/YYY/ZZZ'],
  policy: {
    corsAllow: [
      'https://your-docs.example.com',
      'https://dashboard.yourdomain.com'
    ],
    slackChannel: '#feedback' // optional override
  },
  debounceMs: 30000,
});

Next Steps

Try leaving feedback with the floating button. The nearest heading and selected text will be captured.
